# Homeschool Web Starter

This starter is structured for gradual growth:
- home page to create learners and open the learner hub
- learner hub with preferred learning vehicles first
- subject tiles for Language, Maths, Science, and cross-curricular growth
- language diagnostics page with a reading baseline generator and analyzer

## Current flow
1. Create a learner on the home page.
2. Open the learner hub.
3. Add preferred learning vehicles.
4. Open the Language tile.
5. Generate and analyze the reading baseline.

## Notes
- New learners now start with blank preferred vehicles.
- Preferred vehicles are editable on the learner hub and are stored on the learner record.
- Reading analysis is currently rule-based so the structure stays stable while the project grows.
